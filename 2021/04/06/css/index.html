CSS3有哪些新增的特性？

发表于2021-04-06|更新于2021-04-06
|阅读量:

问题:

CSS3有哪些新增的特性？

答案:

边框

  • border-radius 圆角
  • box-shadow 阴影
  • border-image 边界图片

圆角

  • border-radius

背景

  • background-image 背景图
  • background-size 背景大小
  • background-origin 指定了背景图像的位置区域:content-box, padding-box,和 border-box区域内可以放置背景图像
  • background-clip 背景剪裁属性是从指定位置开始绘制:content-box, padding-box,和 border-box

渐变

  • linear-gradient 线性渐变
  • radial-gradient 径向渐变

文本效果

  • text-shadow 文本阴影
  • box-shadow 盒模型阴影
  • text-overflow 文本溢出时的显示方式
  • word-wrap 文本换行
  • word-break 单词拆分

2D转换

  • translate 位移
  • rotate 旋转
  • scale 缩放
  • skew 倾斜
  • matrix 旋转,缩放,平移,倾斜

3D转换

  • rotateX X轴旋转
  • rotateY Y轴旋转

过渡
-transition

动画

  • @keyframe
  • animation

盒模型

  • box-sizing 设置是否包含padding和border

弹性盒子

  • display:flex

多媒体查询

  • @media
